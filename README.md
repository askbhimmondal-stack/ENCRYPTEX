# Encryptex
A simple, browser-based tool to encrypt and decrypt messages using various algorithms.

Educational project to explore how ciphers work – everything runs client-side in your browser.

## Supported Methods
- Caesar Cipher (shift any number)
- Vigenère Cipher (keyword-based)
- ROT13 (classic quick flip)
- Base64 encoding
- (Optional) Simulated AES / substitution ciphers
